Το Eclipse project αυτό είναι ΗΜΙΤΕΛΕΣ και καλείσθε να το συμπληρώσετε με βάση την εκφώνηση.

0. Διαβάστε προσεκτικά τις οδηγίες από το web site του μαθήματος (π.χ., για το πώς να ονομάσετε το δικό σας Project, τι να έχει μέσα, κλπ).

1. Πρακτικά, σας δίδεται μόνο το κομμάτι του front-end (στο πακέτο client).
Δεν χρειάζεται / επιτρέπεται να πειράξετε τίποτα στο κομμάτι αυτό.

2. Από το back-end, σας δίδεται αρχικα το ΣΥΜΒΟΛΑΙΟ μεταξύ front-end & back-end:
   συγκεκριμένα, το interface IDatasetManager @package server.
   Αυτό είναι το κομμάτι που πρέπει να αναπτύξετε εσείς!
   Δεν επιτρέπεται να πειράξετε τίποτα στο interface IDatasetManager.
   
   Σας δίδεται επίσης και το factory DatasetManagerFactory.
   ΠΡΟΣΟΧΗ: _θέλει διόρθωση_, προφανώς δεν παίζει έτσι (αν το διαβάζετε αυτό στην αρχή του μαθήματος, θα δείτε πώς δουλεύουμε με factories στη διάρκεια του μαθήματος)
   
   Τέλος, σας δίδεται το package utils με έναν _άδειο_ αναγνώστη CSV αρχείων. 
   Έχει οδηγίες για το πού μπορείτε να αντιγράψετε κώδικα για να τον υλοποιήσετε, και ΝΑΙ, μπορείτε να επαναχρησιμοποιήσετε κώδικα που θα βρείτε στο internet για το εν λόγω υποσύστημα. 
   
   	
3. Στο φάκελο Resources έχει διάφορα csv αρχεία για να μπορείτε να δουλέψετε.
   Μπορείτε να τα δείτε εύκολα από οποιοδήποτε text editor (αλλά και από spreadsheets -- προσέξτε μόνο μήπως εκεί πρέπει να μετατρέψετε τα ',' σε ';' για να το δείτε.)
